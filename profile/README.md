## 문제 정의

> **늘어난 규제, 쏟아지는 증빙 — 사람이 수동으로 검토할 수 있는 임계치를 넘다**

HD현대중공업은 약 **3,354개 협력사**의 ESG·안전보건·컴플라이언스 증빙을 관리해야 합니다.
그러나 규제는 빠르게 강화되고 있고, 기존 **체크리스트 기반 수기 검토**로는 한계에 도달했습니다.

| 규제 동향 | 핵심 내용 |
|:---------|:---------|
| **중대재해처벌법 전면 확대** | 2024년 상시근로자 5인 이상 전 사업장 적용, 위험성평가 인정기준 70점 → 90점 상향 |
| **하도급법 개정** | 2025년 3월 국회 통과 — 부당특약 무효화, 공정위 집중 단속 |
| **EU CSDDD 발효** | 공급망 인권·환경 리스크 실사 의무 법제화, 증빙 기반 실사 체계 필수 |

기존 방식은 **전수 수집 → 전수 수동 열람** 으로 반복적이고, 리스크 누락 가능성이 높으며, 판단 병목으로 처리가 지연됩니다.

## 3개 도메인, 하나의 플랫폼

SmartChain은 **안전보건 · 컴플라이언스 · ESG** 세 가지 도메인을 하나의 플랫폼에서 통합 관리합니다.
각 도메인은 완전히 독립된 분석 엔진으로 분리되어, 실무 기준에 맞춘 검증이 가능합니다.

<table>
<tr>
<td width="33%" align="center">

### 안전보건 (Safety)

중대재해처벌법 대응
경영책임자 안전·보건 확보의무 요구

**필수 슬롯**: 안전교육 이수현황, 소방시설 점검표, 위험성 평가서, 안전보건 관리체계

**교차검증**: 출석부 인원수 ↔ 교육인원 인원수

</td>
<td width="33%" align="center">

### 컴플라이언스 (Compliance)

개인정보보호법·하도급법 대응
접대/상품권/ABAC/공정거래 위반 리스크

**필수 슬롯**: 표준 계약서, 개인정보보호 교육, 공정거래위자율점검표

**교차검증**: 교육출석부 ↔ 교육사진 인원비교

</td>
<td width="33%" align="center">

### ESG

지속가능경영 및 공급망 리스크
가치사슬 전반의 인권·환경 점검

**필수 슬롯**: 전기 사용량, 가스 사용량, 수도 사용량, MSDS, 유해물질 재고, 폐기물 대장, 윤리 서약서

**교차검증**: 월별 사용량 ↔ 청구서 금액 매칭

## 서비스 워크플로우

```
협력사 파일 업로드 ──→ AI 판단 ──→ 위험군 분류 ──→ 원청 조치
        │                  │              │              │
   PDF, JPG, XLSX     AI 엔진이        대시보드에서      담당자가 고위험
   증빙 파일을        슬롯/항합성/규칙   위험도(고/중/저)  항목만 집중 처리,
   업로드합니다       자동 검증합니다     유형별로 분류     보완요청서 자동 생성
```

### One-Stop Workflow: 역할별 권한 분리

한 시스템에서 원청 결재 + 협력사 소통이 동시에 이루어집니다.

| 역할 | 대상 | 주요 액션 |
|:-----|:-----|:---------|
| **기안자** (Drafter) | 협력사 담당자 | 증빙 파일 업로드 → AI 슬롯 자동 탐지 · 누락 사전 안내 → 검증 패키지 생성 → 제출 |
| **결재자** (Approver) | 협력사 팀장 | AI 리스크 등급 확인 + why 요약 → 승인/반려로 품질 게이트 통과 여부 결정 |
| **수신자** (Reviewer) | 원청 HD현대 | 제출 결과 조회 → 슬롯별 판정(통과/조치 필요/확인 필요) 확인 → AI 보완요청 자동 생성 |

## AI 기술

### 6단계 검증 파이프라인

파일 업로드부터 최종 리스크 등급 산정까지, 인간의 개입 없이 이루어지는 End-to-End 자동화 프로세스입니다.

```
┌──────────┐   ┌──────────────────┐   ┌──────────┐   ┌──────────┐   ┌──────────────┐   ┌──────────┐
│  1단계   │   │     2단계        │   │  3단계   │   │  4단계   │   │    5단계     │   │  6단계   │
│입력데이터│──→│데이터 추출+AI검증│──→│ 규칙검증 │──→│ 교차검증 │──→│ 보완요청 생성│──→│ 최종집계 │
│  확인    │   │                  │   │          │   │          │   │              │   │          │
└──────────┘   └──────────────────┘   └──────────┘   └──────────┘   └──────────────┘   └──────────┘
  확장자 기반      파일별 데이터 추출     슬롯별 사유코드    도메인별 슬롯 쌍끼리   문제 슬롯 사유코드     규칙 기반 리스크 산출
  파일 분류 +      (텍스트, 날짜, 서명)    그루핑 +          데이터 교차 비교 +     + 한국어 번역 조합 →   + LLM 종합 수정 방향
  슬롯 매핑        + LLM 이상징후 탐지     Verdict 판정      불일치 발견 시 추가    LLM 보완요청 생성      risk_level + verdict
                                                             사유코드 부여                              + why 출력
```

### 목적에 맞는 AI 모델 조합

운영 비용 최소화와 추론 정확도 극대화를 위해, 기능별로 최적의 모델을 분업시켰습니다.

| 모델 | 역할 | 선정 이유 |
|:-----|:-----|:---------|
| **Clova OCR** | 스캔 PDF, 사진 속 텍스트 추출 (한국어 문서) | 한국어 인식률 98%+, 비용 효율 |
| **GPT-4o-mini** | 파일 분류, 슬롯 매핑, 텍스트 분석, 규칙 검증, 보완요청 생성 | 빠른 처리(2~3초), 저렴한 비용 |
| **GPT-5.1 Vision** | 현장 사진 해석 — 서명/날인 확인, 해상도 검증 | 멀티모달 지원, 최고 수준 정확도(98%+) |
| **YOLO26n** | 작업자 인원수 카운팅 (출석부 교차 검증) | 2.4M 경량 모델, 초고속 처리(30FPS), 비용 Zero |

### 부가 AI 기능

| 기능 | 설명 |
|:-----|:-----|
| **법령 확인 챗봇** | RAG 기반 — 사내 규정/법령을 벡터 DB에 구축, 질문 1번으로 관련 조항과 근거 하이라이트 제공 |
| **외부 리스크 탐지** | 뉴스 데이터(GDELT/RSS) 실시간 수집 → 협력사별 부정 신호 점수화 → LOW/MEDIUM/HIGH 위험도 대시보드 |
